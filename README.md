# UnlimitedRadioBattery
Simple and light Plugin for Unlimited Radio Battery in SCP:SL.

# Config:
| Name | Type | Description | Default |
| ---- | ---- | ---- | ---- |
| IsEnabled | bool | Is the Plugin enabled | true |

## Default Config
```yml
using_radio_battery:
  is_enabled: true
```
